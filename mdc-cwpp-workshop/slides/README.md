## Design

- **Aspect**: 16:9 (13.33" × 7.5")
- **Palette**: Midnight Executive — navy primary (`#1E2761`), ice blue (`#CADCFC`), coral accent (`#F96167`), gold spotlight (`#F9C24A`), soft gray surface (`#F2F4FA`)
- **Type**: Georgia for headers, Calibri for body, Consolas for code
- **Pattern**: dark cover + section dividers; light content with surface cards and a 0.08″ coral pip beside each title

## Customising

The deck is generated by `slides/build_pptx.py` (committed as part of the workshop history, but removed from the working tree after build). To regenerate or tweak:

```bash
pip install python-pptx lxml
git show HEAD:slides/build_pptx.py > slides/build_pptx.py
python slides/build_pptx.py
```

Common tweaks:

- **Re-brand**: change the `NAVY`/`ICE`/`CORAL` constants at the top.
- **Re-title**: edit the `title_slide()` call near the bottom.
- **Change agenda**: edit the `agenda` list.
- **Hide a plan**: comment the matching `plan_slide(...)` call.
- **Re-arrange demo**: edit the `steps` list on the kill-chain slide.

## Speaker notes

Speaker notes are not in this version — content density on each slide is enough to talk from. If you want full notes:

1. Open the deck in PowerPoint or Keynote.
2. Use the talking-point bullets from `RUNBOOK.md` § 20 — they map directly to the demo storyboard slide.
3. Insert "Demo →" markers on slides 22 and 26 where you intend to switch out to the lab/Sentinel.
